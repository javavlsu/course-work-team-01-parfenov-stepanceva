# Описание работы 

### SecurityConfig
Отключает CSRF и сессии.
Настраивает доступ к ресурсам апи.

### JwtAuthenticationFilter
Перехватывает http запросы и в куки смотрит accessTokenKanban
Если токен валиден, устанавливает пользователя в контекст безопасности (SecurityContextHolder), разрешая доступ к защищенным ресурсам.

### JwtService
Генерирует рефреш и аксес токены.
Извлекает email из токена (передает в CustomUserDetailsService).
Смотрит подпись и срок действия.

### CustomUserDetailsService
Присваивает дефолтную роль юзер в системе.
Ищет пользователя по email в юзер репозитории.
